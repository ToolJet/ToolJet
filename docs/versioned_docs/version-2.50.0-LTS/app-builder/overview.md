# App-Builder: Overview

ToolJet's App Builder allows you to build applications. ToolJet's app builder has the following major components:

- **[Topbar](/docs/2.50.0-LTS/app-builder/topbar)**: Configure app settings such as application name, layout, app environment, version management, Gitsync and much more.
- **[Canvas](/docs/2.50.0-LTS/app-builder/canvas)**: Arrange and customize components to build the user interface of your app.
- **[Left-sidebar](/docs/2.50.0-LTS/app-builder/left-sidebar)**: Manage [pages](/docs/2.50.0-LTS/tutorial/pages), [inspect](/docs/2.50.0-LTS/how-to/use-inspector) components, queries, or variables, and [debug](/docs/2.50.0-LTS/app-builder/left-sidebar#debugger) errors.
- **[Components library](/docs/2.50.0-LTS/app-builder/components-library)**(right sidebar): Drag and drop components, modify their properties, and style them.
- **[Query Panel](/docs/2.50.0-LTS/app-builder/query-panel)**: Create, edit, and manage queries from various **[datasources](/docs/data-sources/overview)**.
- **[Preview](/docs/2.50.0-LTS/app-builder/preview)**: Preview your application.
