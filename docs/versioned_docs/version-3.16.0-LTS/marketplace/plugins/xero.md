The Xero Plugin for ToolJet enables the integration between ToolJet applications and Xero, a leading cloud-based accounting platform. Using this plugin, you can securely connect to Xero APIs and perform operations related to accounting, payroll, projects, assets, files, and more directly from your ToolJet application.

## Connection

You will need the following credential to connect with Xero :
 - **Client ID**
 - **Client Secret**
  - **Scope**

## Supported Operations

- **App Store**
- **Finance**
- **Accounting**
- **identity**
- **Payroll AU**
- **Payroll UK**
- **Payroll NZ**
- **Projects**
- **Assets**
- **Bank Feeds**
- **Files**
