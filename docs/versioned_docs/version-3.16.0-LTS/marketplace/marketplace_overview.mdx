The ToolJet Marketplace provides a collection of plugins that extend ToolJet's integration capabilities. These plugins connect ToolJet to external services like AI providers, databases, cloud platforms, and business applications.

- **ToolJet Cloud**: All plugins come pre-installed. Configure the credentials and start using them in your applications.
- **Self-hosted**: Install plugins from the Marketplace, then configure credentials to use them.

<img className="screenshot-full img-full" src="/img/marketplace/overview/marketplace-v2.png" alt="Marketplace Overview" />

## Installing a Plugin (Self-Hosted Only)

1. Click the **Settings** icon at the bottom-left of the dashboard.
2. Select **Marketplace** from the menu.
3. Navigate to the **Marketplace** tab to view all available plugins.
4. Click the **Install** button on any plugin card. Once installed, the button changes to **Installed**.

<img className="screenshot-full img-full" src="/img/marketplace/overview/allplugins-v2.png" alt="List of All Plugins" />

## Configuring Plugins

After installing a plugin (or for pre-installed plugins on ToolJet Cloud), configure its credentials:

1. Go to the **Data sources** tab in the dashboard.
2. Scroll down to the **Plugins** section to see available plugins.
3. Click on a plugin to configure its connection credentials.
4. Once configured, the plugin appears in the Query Panel when creating new queries.

<img className="screenshot-full img-full" src="/img/marketplace/overview/installed-plugins.png" alt="Installed plugins" />

## Removing a Plugin (Self-Hosted Only)

:::warning
Removing a plugin will delete all queries associated with it from your applications. If any queries are linked to the plugin, you'll need to remove those queries first before uninstalling.
:::

1. Go to **Settings** > **Marketplace**.
2. Navigate to the **Installed** tab.
3. Click **Remove** on the plugin you want to uninstall.
