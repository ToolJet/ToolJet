ToolJet Workflows supports importing external JavaScript Libraries to perform various operations inside the workflow. By importing external JS libraries you can:
- Extend workflow capabilities beyond built-in nodes.
- Simplify complex data transformations or formatting.
- Validate or sanitize input data efficiently.
- Handle advanced math, string, or statistical computations.

## Importing External Libraries 

1. Create a new workflow.
2. Go to Libraries tab on the left panel of workflow builder, and click on **+ Add library**. <br/>
    <img className="screenshot-full img-s" style={{ marginTop: '15px' }} src="/img/workflows/imp-ext-lib/lib-panel.png" alt="Workflows Library Panel" />
3. After that you will see a modal listing popular JS libraries, you can select the required libraries and click on **add selected libraries** button. <br/>
    <img className="screenshot-full img-l" style={{ marginTop: '15px' }} src="/img/workflows/imp-ext-lib/popular-lib.png" alt="Workflows Library Panel" />
4. By this you have added the library after which you can configure the setup script to start using the library in the workflow.
