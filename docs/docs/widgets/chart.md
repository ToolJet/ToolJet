# Chart

Chart widget takes the chart type, data and styles to draw charts using Plotly.js.

Support chart types:

- Line charts
- Bar charts
- Pie charts

## Line charts

Data requirements:

The data needs to be an array of objects and each object should have `x` and `y` keys.

Example:

```
[
  { "x": "Jan", "y": 100},
  { "x": "Feb", "y": 80},
  { "x": "Mar", "y": 40}
]
```
