The ToolJet API lets you manage users, workspaces, applications, modules, and groups programmatically. Use it to integrate ToolJet into your existing identity management workflows, automate user provisioning, and manage application lifecycle from outside the UI.

## Base URL

```
https://{your-domain}/api/ext
```

## Endpoints

| Category | Endpoints |
|:---------|:----------|
| [Users](/api/tooljet-api/users/get-all-users) | Get, create, update, and manage user workspace relations and metadata |
| [Workspaces](/api/tooljet-api/workspaces/get-all-workspaces) | List workspaces and app details |
| [Apps](/api/tooljet-api/apps/export-application) | Export and import applications |
| [Modules](/api/tooljet-api/modules/list-modules) | List, export, and import modules |
| [Groups](/api/tooljet-api/groups/create-group) | Create, read, update, and delete workspace groups |
| [ToolJet Database](/api/tooljet-api/tooljet-db/export-table-data) | Export ToolJet Database table data as a CSV file |
