---

<EndpointHeader method="GET" path="/user/{identifier}" />

<ApiColumns>
<ApiLeft>

Returns details of a user identified by UUID or email address.

:::note
If the identifier matches no user, the endpoint returns `200` with an empty array `[]` — no `404` is thrown.
:::

<ParamsTable title="Path Parameters" params={[{"name":"identifier","type":"string","required":true,"description":"UUID (e.g. `d1493ea9-...`) or email (e.g. `john@example.com`).","in":"path"}]} />

]} />

</ApiLeft>
<ApiRight>
