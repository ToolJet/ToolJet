# API Reference

ToolJet exposes a set of REST APIs that let you programmatically manage users, workspaces, applications, modules, and more. All APIs are secured and require an access token passed in the `Authorization` header.

## Authentication

All API requests (except SCIM, which uses its own token) require an `Authorization` header with a `Basic` access token configured in your `.env` file.

```bash
Authorization: Basic <access_token>
```

To enable the ToolJet API, add these variables to your `.env`:

| Variable | Description |
|:---------|:------------|
| `ENABLE_EXTERNAL_API` | `true` or `false` |
| `EXTERNAL_API_ACCESS_TOKEN` | Your access token |
