---
id: add-git-config
title: "Add GitHub HTTPS Git Configuration"
---

<EndpointHeader method="POST" path="/organizations/git" />

Configure GitHub HTTPS settings for an organization by associating a GitHub App and repository.






<ParamsTable title="Request Body" params={[{"name":"organizationId","type":"string","required":true,"description":"UUID of the organization.","in":"body"},{"name":"gitUrl","type":"string","required":true,"description":"Full HTTPS Git URL.","in":"body"},{"name":"branchName","type":"string","required":true,"description":"Branch to sync with (e.g. \"main\").","in":"body"},{"name":"githubAppId","type":"string","required":true,"description":"GitHub App ID.","in":"body"},{"name":"githubAppInstallationId","type":"string","required":true,"description":"GitHub App installation ID.","in":"body"},{"name":"githubAppPrivateKey","type":"string","required":true,"description":"GitHub App private key in PEM format.","in":"body"}]} />

<RequestExample code={`curl -X POST "https://{your-domain}/api/ext/organizations/git" \
  -H "Authorization: Basic <access_token>" \
  -H "Content-Type: application/json" \
  -d '{
  "organizationId": "45892c81-c1f0-48c6-8875-c2e4fca516f8",
  "gitUrl": "https://github.com/username/repository.git",
  "branchName": "main",
  "githubAppId": "123456",
  "githubAppInstallationId": "78910",
  "githubAppPrivateKey": "-----BEGIN RSA PRIVATE KEY-----\n...\n-----END RSA PRIVATE KEY-----"
}'`} />

**Response:** `201`
