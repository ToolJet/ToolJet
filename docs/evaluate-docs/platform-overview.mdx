# Platform Overview

ToolJet is an open-source low-code platform for building internal tools. Engineering teams use it to ship admin panels, operational dashboards, approval workflows, and data management applications in days instead of months.

## The Problem It Solves

Internal tools are expensive to build and maintain. A custom admin panel takes a senior engineer 2–3 weeks from scratch, another week to wire up authentication and permissions, and an ongoing commitment to maintain as the underlying data schema evolves. Multiply that across 10–15 internal tools and you're looking at one or two engineers perpetually occupied with tooling that doesn't ship product.

ToolJet compresses that cycle. A developer connects to a data source, drags in a table and a form, wires up queries, configures permissions, and ships. The same tool takes hours, not weeks. Operations, finance, and support teams can then maintain and extend those apps without filing engineering tickets for every change.

## Who Uses ToolJet

- **Platform and infrastructure teams** building internal tooling infrastructure: user management consoles, configuration dashboards, deployment controls, incident response tools.
- **Operations and data teams** needing to act on live data without waiting for engineering cycles: approval queues, case management, reporting dashboards, data quality tooling.
- **Engineering-enabled product teams** iterating on internal workflows before committing to a custom build: intake forms, routing logic, notification systems.
