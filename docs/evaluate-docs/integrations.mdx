# Integrations & Data Sources

ToolJet apps query your existing data directly — databases, APIs, and SaaS tools — without ETL pipelines or middleware. This section covers what's supported, how the query layer works, and how credentials are handled.

## ToolJet Database

Every workspace includes a built-in Postgres-backed database — no external database required for lightweight use cases.

- Full table management through a GUI: create tables, add columns, set constraints
- Query it like any other data source from the query panel
- Accessible via REST API for external systems
- Appropriate for small-to-medium datasets and prototyping; for high-volume transactional data, connect your own database

## Connector Marketplace

The marketplace lets teams publish and install custom connectors. Organizations with internal APIs or proprietary tools can build a connector once and make it available to all apps in the workspace.

Marketplace connectors follow the same credential model and query panel interface as built-in connectors.
