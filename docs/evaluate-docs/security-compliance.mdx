# Security & Compliance

*This section covers ToolJet's security model and compliance certifications: what data ToolJet touches, how credentials are protected, what auditors have reviewed, and what controls are available for regulated environments.*

**Coming soon.** Content is being drafted for this section.

Topics this section will cover:

- Compliance certifications — SOC 2 Type II, GDPR, ISO 27001
- The data proxy model — ToolJet routes queries server-side but never stores result data; data flows from your source directly to the user's browser
- Credential storage — AES-256-GCM encryption for all datasource credentials at rest
- Transport security — TLS 1.2+ for all connections in transit
- Audit logs (Enterprise) — comprehensive activity logging for all user actions; exportable to SIEM tools
- Network controls — IP whitelisting, VPC deployment, private networking options
- Vulnerability disclosure and responsible security practices
- ToolJet Trust Center

In the meantime, see [Security & Monitoring](/docs/security/compliance) for technical reference.
